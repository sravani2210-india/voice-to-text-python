# Voice to Text using Python

This project converts spoken voice input into text using speech recognition.

## Technologies Used
- Python
- SpeechRecognition Library

## How it Works
The program listens to audio from the microphone and converts it into text using Google's speech recognition API.

## How to Run

Install required libraries:

pip install SpeechRecognition
pip install pyaudio

Run the program:

python voice_to_text.py
